# GRU Quiz

## True/False

### Q1: GRU has more gates than LSTM.
**Answer:** False - GRU has 2 gates, LSTM has 3.

### Q2: GRU is faster than LSTM.
**Answer:** True - Fewer gates means fewer computations.

### Q3: GRU has a separate cell state.
**Answer:** False - GRU only has a hidden state.

### Q4: The update gate controls how much past info to keep.
**Answer:** True - z_t close to 1 keeps more past info.

### Q5: GRU always outperforms LSTM.
**Answer:** False - Performance depends on the task.

## Multiple Choice

### Q6: How many gates does GRU have?
A) 1  B) 2  C) 3  D) 4
**Answer:** B

### Q7: What does the reset gate do?
A) Controls new info  B) Controls what to forget  C) Controls output  D) None
**Answer:** B

### Q8: GRU combines which gates?

**Answer:** A

### Q9: GRU vs LSTM parameters?

**Answer:** B
