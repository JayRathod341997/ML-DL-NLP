# GAN Quiz

## True/False

### Q1: Generator creates fake samples.
**Answer:** True.

### Q2: Discriminator is always correct.
**Answer:** False - generator tries to fool it.

### Q3: Mode collapse is desirable.
**Answer:** False - it's a training problem.

### Q4: GAN training is always stable.
**Answer:** False - notoriously unstable.

## Multiple Choice

### Q5: Generator input is:
A) Real data  B) Random noise  C) Labels  D) None
**Answer:** B

### Q6: Discriminator output is:
A) Image  B) Probability  C) Loss  D) Vector
**Answer:** B

### Q7: GAN stands for:
A) Generative Adversarial Network  B) General Adversarial Network  C) Generated Adversarial Network  D) None
**Answer:** A

### Q8: Training is:
A) Cooperative  B) Adversarial  C) Random  D) Supervised only
