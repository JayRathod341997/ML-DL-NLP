# Topic Modeling - Exercises

## Exercise 1: Basic LDA
Apply LDA to a corpus of 5 documents and extract 2 topics.

## Exercise 2: Compare LDA vs NMF
Compare the topics generated by LDA and NMF on the same corpus.

## Exercise 3: Optimal Topics
Find the optimal number of topics using coherence score.

## Exercise 4: Topic Visualization
Visualize topics using word clouds or bar charts.

## Exercise 5: Document Classification
Use topic distributions as features for document classification.
