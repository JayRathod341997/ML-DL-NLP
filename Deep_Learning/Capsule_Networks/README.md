# Capsule Networks

## Overview

Capsule networks use capsules (groups of neurons) that represent instantiation parameters, addressing limitations of pooling.

### Key Characteristics

- **Capsules**: Group of neurons representing entities
- **Dynamic Routing**: Capsules communicate
- **Pose Information**: Preserve spatial relationships
