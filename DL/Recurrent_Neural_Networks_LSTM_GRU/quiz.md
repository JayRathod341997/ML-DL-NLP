# RNN / LSTM / GRU - Quiz

## Fill in the Blanks

1. RNNs maintain a ________ state across time steps.
2. Vanishing gradients make it hard to learn ________ dependencies.
3. LSTMs use gates to ________ and ________ information.
4. GRUs are generally ________ than LSTMs (fewer gates).
5. Transformers rely on ________ instead of recurrence.

## True or False

1. RNNs process sequences in parallel by default.  
2. LSTMs were designed to mitigate vanishing gradients.

4. Sequence order matters for RNNs.
