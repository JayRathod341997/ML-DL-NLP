# Transformers - Quiz

## Fill in the Blanks

1. Transformers rely heavily on ________ mechanisms.
2. Positional encoding provides information about token ________.
3. Multi-head attention uses multiple ________ in parallel.
4. Standard attention complexity is ________ in sequence length.
5. Self-attention uses queries, keys, and values from the ________ sequence.

## True or False

1. Transformers are naturally order-aware without positional encodings.  
2. Transformers can be trained in parallel across time steps.  
3. Multi-head attention allows different representation subspaces.  
4. Attention always improves performance regardless of data quality.  

## Answers

**Fill in the Blanks**
1. attention
2. order/position
3. heads
4. quadratic (O(n^2))
5. same

**True or False**
1. False
2. True
3. True
4. False
