# Random Forest - Quiz

## Fill in the Blanks

1. Random Forest is an ensemble of ________ Decision Trees.
2. Each tree in Random Forest is trained on a random subset of data called ________.
3. The process of taking random samples is called ________.
4. Random Forest uses random selection of features at each ________.
5. The final prediction is made by ________ vote.

## True or False

1. Random Forest always uses all features for each tree.
2. Random Forest reduces overfitting compared to a single Decision Tree.
3. Random Forest can handle missing values.
4. More trees always give better results.
5. Random Forest can be used for both classification and regression.

## Multiple Choice

1. What is the main purpose of Random Forest?
   a) Reduce the number of features
   b) Improve accuracy and reduce overfitting
   c) Speed up training
   d) Reduce memory usage

2. What is bootstrapping in Random Forest?
   a) Training on the entire dataset
   b) Random sampling with replacement
   c) Feature selection
   d) Pruning trees

3. How many trees are typically used in Random Forest?
   a) 1
   b) 10
   c) 100 or more
   d) Unlimited

## Answers

**Fill in the Blanks:**
1. multiple
2. bootstrap sample
3. bagging/bootstrap
4. split/node
5. majority

**True or False:**
1. False - Uses random subset of features
2. True
3. True
4. False - Diminishing returns after certain point
5. True

**Multiple Choice:**
1. b
2. b
3. c
