# CNN - Quiz

## Fill in the Blanks

1. CNNs use ________ to scan local parts of an image.
2. Parameter sharing means the same filter is used at every ________ location.
3. Pooling reduces the ________ dimensions of feature maps.
4. A feature map is produced after applying a ________ operation.
5. CNNs are especially good for ________ data.

## True or False

1. CNN filters are learned during training.  
2. Pooling always increases model parameters.  
3. CNNs capture local spatial patterns.  
4. A fully-connected layer preserves spatial structure better than convolution.  

## Multiple Choice

1. Which layer typically reduces spatial resolution?
   a) Convolution  
   b) Pooling  
   c) Embedding  
   d) Softmax  

2. What is a receptive field?
   a) Training set size  
   b) The input region influencing an activation  
   c) The number of classes  
   d) The optimizer choice  

## Answers

**Fill in the Blanks**
1. filters/kernels
2. spatial
3. spatial
4. convolution
5. image/grid

**True or False**
1. True
2. False
3. True
4. False

**Multiple Choice**
1. b
2. b
