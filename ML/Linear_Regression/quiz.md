# Linear Regression - Quiz

## Fill in the Blanks

1. Linear Regression is used to predict a ________ value.
2. The formula for linear regression is Y = ________ X + b.
3. The "m" in Y = mX + b is called the ________.
4. The "b" in Y = mX + b is called the ________.
5. R-squared measures how well the model ________ the data.

## True or False

1. Linear Regression can be used for classification problems.
2. The goal of Linear Regression is to minimize the sum of squared errors.
3. Linear Regression assumes a linear relationship between variables.
4. You can use Linear Regression to predict house prices.
5. A high R-squared always means a good model.

## Multiple Choice

1. What does the slope (m) represent in Y = mX + b?
   a) The starting point
   b) How much Y changes when X changes by 1
   c) The error in prediction
   d) None of the above

2. What is the range of R-squared?
   a) 0 to 1
   b) -1 to 1
   c) 0 to 100
   d) Any number

3. Which metric is NOT used to evaluate Linear Regression?
   a) R-squared
   b) Mean Squared Error
   c) Accuracy
   d) RMSE

## Answers

**Fill in the Blanks:**
1. continuous/numerical
2. m
3. slope/coefficient
4. intercept
5. fits/explains

**True or False:**
1. False - It's for regression (predicting numbers)
2. True
3. True
4. True
5. False - A high R-squared could indicate overfitting

**Multiple Choice:**
1. b
2. a
3. c
