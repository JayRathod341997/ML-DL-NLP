# Logistic Regression - Quiz

## Fill in the Blanks

1. Logistic Regression is used for ________ problems.
2. The output of Logistic Regression is a ________ between 0 and 1.
3. The ________ function converts any value to a probability.
4. The decision boundary in Logistic Regression is ________.
5. If probability > 0.5, we predict ________.

## True or False

1. Logistic Regression can predict continuous values.
2. The sigmoid function always outputs values between 0 and 1.
3. Logistic Regression is a type of classification algorithm.
4. You can use Logistic Regression for multi-class problems.
5. Maximum Likelihood Estimation is used to train Logistic Regression.

## Multiple Choice

1. What is the threshold probability for classification in Logistic Regression?
   a) 0.3
   b) 0.5
   c) 0.7
   d) 1.0

2. Which function is used in Logistic Regression?
   a) Linear
   b) Sigmoid/Logistic
   c) ReLU
   d) Tanh

3. What does the log-odds (logit) represent?
   a) Probability / (1 - Probability)
   b) Log of probability
   c) Square root of probability
   d) Probability squared

## Answers

**Fill in the Blanks:**
1. classification
2. probability
3. sigmoid/logistic
4. linear (for linear case)
5. class 1/positive

**True or False:**
1. False - It predicts classes (0 or 1)
2. True
3. True
4. True (using One-vs-Rest or Softmax)
5. True

**Multiple Choice:**
1. b
2. b
3. a
