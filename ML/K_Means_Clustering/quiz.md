# K-Means Clustering - Quiz

## Fill in the Blanks

1. K-Means is a ________ learning algorithm.
2. In K-Means, K represents the number of ________.
3. The center of a cluster is called a ________.
4. K-Means tries to minimize ________ distance.
5. The algorithm starts with ________ centroids.

## True or False

1. K-Means requires the number of clusters to be specified.
2. K-Means always finds the global optimum.
3. K-Means is sensitive to initialization.
4. K-Means can only use Euclidean distance.
5. K-Means works well with non-spherical clusters.

## Multiple Choice

1. What is the Elbow method used for?
   a) Choosing the number of clusters
   b) Initializing centroids
   c) Computing distances
   d) Updating centroids

2. What happens in one iteration of K-Means?
   a) Assign points to clusters and update centroids
   b) Only assign points to clusters
   c) Only update centroids
   d) Calculate distances between all points

3. What is the objective function in K-Means?
   a) Maximize inter-cluster distance
   b) Minimize intra-cluster distance
   c) Maximize silhouette score
   d) Minimize entropy

## Answers

**Fill in the Blanks:**
1. unsupervised
2. clusters
3. centroid
4. within-cluster/squared Euclidean
5. random/initial

**True or False:**
1. True
2. False - Can get stuck in local optima
3. True
4. False - Can use other distances
5. False - Assumes spherical clusters

**Multiple Choice:**
1. a
2. a
3. b
