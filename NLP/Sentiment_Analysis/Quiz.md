# Sentiment Analysis - Quiz

## Q1: "I love this product!" is?
A) Negative
B) Positive
C) Neutral
D) Mixed

**Answer: B**

## Q2: Which is NOT a sentiment type?
A) Positive
B) Negative
C) Numerical
D) Neutral

**Answer: C**

## Q3: Amazon uses sentiment analysis for?
A) Product recommendations
B) Weather
C) Maps
D) Calculator

**Answer: A**

## Q4: Deep learning approach for sentiment?
A) LSTM only
B) BERT
C) CNN
D) All of above

**Answer: D**
