# Language Models

## Explain Like I'm 5

When you write on your phone and it suggests the next word - that's a language model guessing what you might say next!

Language models learn to predict what words come next based on what you've already typed!

## What is a Language Model?

A probability distribution over sequences of words that predicts the likelihood of a sequence.

## Types:

1. **N-gram Models**: Predict based on previous N-1 words
2. **Neural Language Models**: Use neural networks
3. **Transformer-based**: GPT, LLaMA, Claude

## Where Used?

1. **Autocomplete**: Email, search, messaging
2. **Speech Recognition**: Predict next word in speech
3. **Machine Translation**: Rank translations
4. **Text Generation**: Create human-like text

## Key Concepts:

- **Perplexity**: Measure of how well model predicts
- **Pre-training**: Train on large text corpus
- **Fine-tuning**: Adapt to specific tasks

## Enterprise Example:

**ChatGPT** is a large language model that can answer questions, write code, and have conversations!
