# NER - Quiz

## Q1: What does NER stand for?
A) Name Entity Recognition
B) Named Entity Recognition
C) Named Entity Reference
D) Natural Entity Recognition

**Answer: B**

## Q2: Which is NOT a common entity type?
A) PERSON
B) LOCATION
C) ANIMAL
D) ORGANIZATION

**Answer: C**

## Q3: Which library is best for production NER?
A) NLTK
B) spaCy
C) TextBlob
D) All equal

**Answer: B**

## Q4: NER helps in?
A) Text classification
B) Entity extraction
C) Language generation
D) Translation

**Answer: B**
