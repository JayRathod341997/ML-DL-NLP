# Topic Modeling - Quiz

## Q1: LDA stands for?
A) Latent Dirichlet Allocation
B) Linear Discriminant Analysis
C) Latent Data Analysis
D) Linear Data Allocation

**Answer: A**

## Q2: LDA is a?
A) Supervised learning
B) Unsupervised learning
C) Reinforcement learning
D) Semi-supervised learning

**Answer: B**

## Q3: Which is NOT a topic modeling algorithm?
A) LDA
B) NMF
C) SVM
D) LSA

**Answer: C**

## Q4: Topics in LDA are?
A) Fixed words
B) Distributions over words
C) Single words
D) None

**Answer: B**

## Q5: Number of topics is determined by?
A) Always fixed
B) Domain knowledge, coherence score
C) Random
D) Cannot be determined

**Answer: B**
