# VAE Quiz

## True/False

### Q1: VAE can generate new data.
**Answer:** True - samples from latent distribution.

### Q2: VAE uses KL divergence loss.
**Answer:** True.

### Q3: VAE has deterministic latent space.
**Answer:** False - it's probabilistic.

### Q4: Posterior collapse is desirable.
**Answer:** False - it's a training problem.

### Q5: VAE reconstruction is always perfect.
**Answer:** False - outputs can be blurry.

---

## Multiple Choice

### Q6: VAE learns:
A) Only reconstruction  B) Distribution parameters  C) Classification  D) None
**Answer:** B

### Q7: Latent space is:
A) Random  B) Structured  C) Fixed  D) Empty
**Answer:** B

### Q8: Sampling happens in:
A) Input space  B) Output space  C) Latent space  D) None
**Answer:** C
