# Naive Bayes - Quiz

## Fill in the Blanks

1. Naive Bayes is based on ________ theorem.
2. The "Naive" part assumes features are ________ of each other.
3. Naive Bayes is a ________ learning algorithm.
4. Gaussian Naive Bayes assumes ________ distribution for continuous features.
5. Naive Bayes calculates ________ probability.

## True or False

1. Naive Bayes assumes all features are independent.
2. Naive Bayes can only handle categorical features.
3. Naive Bayes is computationally efficient.
4. Naive Bayes is sensitive to irrelevant features.
5. Naive Bayes can be used for multi-class classification.

## Multiple Choice

1. Which Naive Bayes variant is used for text classification?
   a) Gaussian
   b) Multinomial
   c) Bernoulli
   d) Categorical

2. What is the main advantage of Naive Bayes?
   a) High accuracy
   b) Speed and simplicity
   c) Handles missing values well
   d) Works with small data

3. What does the "Bayes" part refer to?
   a) Bayes theorem
   b) Bayes error
   c) Bayes network
   d) Bayes classifier

## Answers

**Fill in the Blanks:**
1. Bayes'
2. independent
3. supervised
4. Gaussian/normal
5. posterior/conditional

**True or False:**
1. True
2. False - Has Gaussian, Multinomial, Bernoulli variants
3. True
4. False - Actually robust to irrelevant features
5. True

**Multiple Choice:**
1. b
2. b
3. a
