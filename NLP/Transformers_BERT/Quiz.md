# Transformers & BERT - Quiz

## Q1: When was Transformer introduced?
A) 2015
B) 2016
C) 2017
D) 2018

**Answer: C**

## Q2: BERT is?
A) Unidirectional
B) Bidirectional
C) Left-to-right only
D) Right-to-left only

**Answer: B**

## Q3: Self-attention allows?
A) Sequential processing
B) Parallel processing
C) Only looking at one word
D) Ignoring context

**Answer: B**

## Q4: BERT training uses?
A) Only supervised learning
B) Masked Language Modeling
C) Only word2vec
D) None

**Answer: B**

## Q5: Transformer has?
A) Encoder only
B) Decoder only
C) Both encoder and decoder
D) Neither

**Answer: C**
