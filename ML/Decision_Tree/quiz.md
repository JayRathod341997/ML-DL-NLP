# Decision Tree - Quiz

## Fill in the Blanks

1. A Decision Tree makes decisions by asking a series of ________ questions.
2. The top node of a Decision Tree is called the ________.
3. The endpoints of a Decision Tree are called ________.
4. The process of splitting nodes is called ________.
5. Preventing the tree from becoming too complex is called ________.

## True or False

1. Decision Trees can handle both numerical and categorical data.
2. Decision Trees are prone to overfitting.
3. Decision Trees require feature scaling.
4. The root node represents the most important feature.
5. Decision Trees can only be used for classification.

## Multiple Choice

1. Which criterion is used to split nodes in Classification trees?
   a) Mean Squared Error
   b) Gini Impurity / Entropy
   c) Variance
   d) Correlation

2. What is the maximum depth of a Decision Tree?
   a) 1
   b) 2
   c) It depends on the parameter set
   d) Unlimited

3. What is pruning in Decision Trees?
   a) Adding more nodes
   b) Removing nodes to prevent overfitting
   c) Training the tree
   d) Making predictions

## Answers

**Fill in the Blanks:**
1. yes/no
2. root
3. leaves
4. splitting/pruning
5. regularization/pruning

**True or False:**
1. True
2. True
3. False - No scaling needed
4. True
5. False - Can be used for regression too

**Multiple Choice:**
1. b
2. c
3. b
