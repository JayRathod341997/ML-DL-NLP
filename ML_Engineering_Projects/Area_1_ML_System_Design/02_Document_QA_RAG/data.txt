# Document Q&A RAG — Public Datasets & Document Sources

## Q&A Evaluation Datasets
# Stanford Question Answering Dataset (SQuAD) — Wikipedia passages + Q&A pairs
# HuggingFace dataset: rajpurkar/squad
https://huggingface.co/datasets/rajpurkar/squad

# Natural Questions — real Google search queries + Wikipedia answers
# HuggingFace dataset: google-research-datasets/natural_questions
https://huggingface.co/datasets/google-research-datasets/natural_questions

# TriviaQA — reading comprehension with web/wiki evidence
# HuggingFace dataset: trivia_qa, config="rc.wikipedia"
https://huggingface.co/datasets/trivia_qa

## Local LLM (via Ollama — no API key needed)
# Install Ollama: https://ollama.ai
# Then pull models:
#   ollama pull llama3.2:3b    (small, runs on CPU, ~2GB)
#   ollama pull mistral:7b     (better quality, needs ~8GB RAM)
#   ollama pull phi3:mini      (very small, fast)
https://ollama.ai/download
