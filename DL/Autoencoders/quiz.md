# Autoencoders - Quiz

## Fill in the Blanks

1. The ________ compresses input into a latent representation.
2. The ________ reconstructs input from the latent code.
3. Autoencoders are often trained with ________ loss.
4. For anomaly detection, anomalies usually have ________ reconstruction error.
5. Denoising autoencoders learn to remove ________ from inputs.

## True or False

1. Autoencoders always require labeled data.  
2. A smaller bottleneck forces more compression.  
3. High capacity autoencoders can reconstruct anomalies well.  
4. Reconstruction error can be used as an anomaly score.  

## Answers

**Fill in the Blanks**
1. encoder
2. decoder
3. reconstruction (e.g., MSE)
4. higher
5. noise

**True or False**
1. False
2. True
3. True
4. True
