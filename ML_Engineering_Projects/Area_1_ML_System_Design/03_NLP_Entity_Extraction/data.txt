# NLP Entity Extraction — Public Datasets

## NER Training & Evaluation Benchmarks
# CoNLL-2003 (gold standard NER: PER, ORG, LOC, MISC)
# HuggingFace dataset: eriktks/conll2003
https://huggingface.co/datasets/eriktks/conll2003

# OntoNotes 5.0 (18 entity types: DATE, MONEY, CARDINAL, GPE, etc.)
# HuggingFace dataset: tner/ontonotes5
https://huggingface.co/datasets/tner/ontonotes5

## Pre-trained NER Models (auto-downloaded via transformers)
# BERT fine-tuned on CoNLL-2003 (PER, ORG, LOC, MISC)
# model_name = "dslim/bert-base-NER"
https://huggingface.co/dslim/bert-base-NER

# DistilBERT for NER (faster, smaller)
# model_name = "elastic/distilbert-base-uncased-finetuned-conll03-english"
https://huggingface.co/elastic/distilbert-base-uncased-finetuned-conll03-english

# RoBERTa large NER (highest accuracy)
# model_name = "Jean-Baptiste/roberta-large-ner-english"
https://huggingface.co/Jean-Baptiste/roberta-large-ner-english
