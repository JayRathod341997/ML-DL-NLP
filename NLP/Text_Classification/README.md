# Text Classification

## Explain Like I'm 5

Imagine you have a pile of emails and you want to sort them into: Inbox, Spam, Social, Promotions!

Text classification is like teaching a computer to automatically sort text into categories!

## What is Text Classification?

Automatically assigning predefined categories to text documents.

## Types:

1. **Binary**: Spam/Not Spam
2. **Multi-class**: News categories (Sports, Politics, Entertainment)
3. **Multi-label**: Multiple labels per document

## Where Used?

1. **Email Filtering**: Spam detection
2. **Sentiment Analysis**: Positive/Negative reviews
3. **Topic Labeling**: Auto-categorize articles
4. **Language Detection**: Identify language

## Approaches:

1. **Traditional**: BoW/TF-IDF + ML classifiers
2. **Deep Learning**: CNN, LSTM
3. **Transformers**: BERT fine-tuning

## Enterprise Example:

**Google Gmail** automatically categorizes emails into Primary, Social, and Promotions tabs using text classification!
