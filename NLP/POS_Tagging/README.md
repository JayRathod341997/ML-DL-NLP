# Part of Speech (POS) Tagging

## Explain Like I'm 5

When you learn grammar in school, you learn that words can be:
- Nouns: names of things (cat, dog, city)
- Verbs: action words (run, eat, think)
- Adjectives: describing words (big, happy, red)
- And more!

POS tagging is like a computer finding out what job each word does in a sentence!

## What is POS Tagging?

Assigning grammatical tags to each word:
- NN: Noun, singular
- VB: Verb
- JJ: Adjective
- RB: Adverb
- DT: Determiner
- IN: Preposition

## Where Used?

1. **Machine Translation**: Understand grammar differences
2. **Text-to-Speech**: Pronunciation
3. **Named Entity Recognition**: Identify entities
4. **Question Answering**: Parse question structure

## Tools:
- spaCy (accurate, fast)
- NLTK (basic)
- Stanford Parser

## Example:
"The cat sat on the mat"
The/DT cat/NN sat/VB on/IN the/DT mat/NN
