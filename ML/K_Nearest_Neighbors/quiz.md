3. KNN does not have a training phase; it is a ________ learner.
4. The distance between points is calculated using a ________ function.

## Multiple Choice

1. What happens when K=1 in KNN?
   a) The model always predicts the majority class
   b) The model predicts based on the nearest neighbor
   c) The model cannot make predictions
   d) The model uses all neighbors

2. Which distance metric is commonly used in KNN?
   a) Euclidean distance
   b) Pearson correlation
   c) Chi-square
   d) ANOVA

3. What is the curse of dimensionality?
   a) Too few features
   b) Distance becomes meaningless in high dimensions
   c) Too many classes
   d) KNN works better with more features
