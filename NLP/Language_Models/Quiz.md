# Language Models - Quiz

## Q1: Language model predicts?
A) Next word probability
B) Image classification
C) Object detection
D) Speech recognition

**Answer: A**

## Q2: Lower perplexity means?
A) Worse model
B) Better model
C) No difference
D) Random

**Answer: B**

## Q3: GPT is?
A) Bidirectional
B) Left-to-right
C) Random
D) None

**Answer: B**

## Q4: Temperature controls?
A) Speed
B) Randomness
C) Accuracy
D) Size

**Answer: B**

## Q5: BERT is?
A) Generative
B) Bidirectional
C) Left-to-right only
D) None

**Answer: B**
