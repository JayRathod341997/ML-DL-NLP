# Perceptron - Quiz

## Fill in the Blanks

1. A perceptron is a ________ neuron model used for ________ classification.
2. The perceptron output is typically produced by a ________ function.
3. A perceptron can only solve problems that are ________ separable.
4. XOR is a classic example of a problem that is ________ linearly separable.
5. The bias term shifts the decision boundary away from the ________.

## True or False

1. A perceptron can learn any boolean function.  
2. The perceptron update uses the error `(y - y_hat)`.  
3. If the dataset is linearly separable, the perceptron can converge.  
4. A perceptron outputs a probability between 0 and 1.  

## Multiple Choice

1. Which problem can a single perceptron solve?
   a) XOR  
   b) AND  
   c) Two-moons  
   d) Spiral classification  

2. What does the learning rate control?
   a) Number of features  
   b) Step size of updates  
   c) Dataset size  
   d) The activation function

**True or False**
1. False
