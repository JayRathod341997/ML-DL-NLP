# NER - Interview Questions

## Q1: What is Named Entity Recognition?
NER identifies and classifies named entities in text into predefined categories like PERSON, ORG, LOC.

## Q2: Common entity types?
PERSON, ORG, LOC, DATE, TIME, MONEY, PERCENT

## Q3: Difference between NER and Entity Linking?
- NER: Identifies and classifies entities
- Entity Linking: Connects entities to knowledge base IDs

## Q4: Approaches to NER?
1. Rule-based (regex, dictionaries)
2. Machine Learning (CRF)
3. Deep Learning (BiLSTM-CRF)
4. Transformers (BERT-based)

## Q5: Challenges in NER?
- Ambiguous entities
- New/unseen entities
- Nested entities
- Domain-specific entities
