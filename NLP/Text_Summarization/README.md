# Text Summarization

## Explain Like I'm 5

Imagine you have a long story and want to tell your friend the most important parts in just a few sentences!

Text summarization does exactly that - creates shorter versions of longer text while keeping the main ideas!

## Types:

1. **Extractive**: Select important sentences/words from original text
2. **Abstractive**: Generate new sentences that capture the meaning

## Where Used?

1. **News Summaries**: Quick news updates
2. **Document Review**: Long reports → key points
3. **Search Results**: Preview documents
4. **Customer Support**: Auto-summarize tickets

## Approaches:

1. **TF-IDF Based**: Score sentences by importance
2. **Sequence-to-Sequence**: Neural networks generate summaries
3. **Transformers**: BERT, T5, BART for abstractive

## Enterprise Example:

**Google News** provides brief summaries of news stories, allowing users to quickly understand the main points!
