# Transformers & BERT

## Explain Like I'm 5

Imagine reading a whole book but remembering everything about every character, plot, and detail - and being able to answer any question about it!

That's what Transformers do! They can understand context from both directions (left and right) at the same time, unlike older models that only looked one way.

## What are Transformers?

Deep learning architecture using "Self-Attention" mechanism:
- Processes entire sequence at once
- Captures long-range dependencies
- Revolutionized NLP in 2017

## BERT (Bidirectional Encoder Representations)

- Bidirectional: reads text both directions
- Pre-trained on large corpus
- Fine-tuned for specific tasks

## Key Concepts:

1. **Self-Attention**: Focus on relevant parts
2. **Transformers**: Encoder + Decoder
3. **Fine-tuning**: Adapt to new tasks

## Where Used?

1. **Question Answering**: Answer questions from context
2. **Text Classification**: Sentiment, spam detection
3. **Named Entity Recognition**: Extract entities
4. **Machine Translation**: Translate languages

## Enterprise Example:

**Google Search** uses BERT to better understand search queries and provide more relevant results!
