# Deep Learning (DL)

This folder contains **Deep Learning algorithms** organized **folder-wise**, similar to the `ML/` structure.

Each algorithm folder includes:
- `README.md` — detailed explanation (starts ELI5, ends with enterprise example), uses, benefits, limitations, and (when helpful) simple visualizations
- `Interview.md` — frequently asked interview questions with answers
- `quiz.md` — concept-focused quiz + answers
- `codes/`
  - `01_in_memory_example.py`
  - `02_csv_example.py`
  - `03_exercise.py`
  - `04_solution.py`
- `dataset/` — one or more `.csv` files used by the code examples

## Notes

- Code examples are written in **Python**.
- Many examples use **PyTorch** (`torch`). If it is not installed, the scripts will tell you what to install.
- Datasets are intentionally tiny so they’re easy to read and run locally.
