# Support Vector Machine (SVM) - Quiz

## Fill in the Blanks

1. SVM finds a ________ that maximizes the margin between classes.
2. Points on the margin are called ________ vectors.
3. The kernel trick allows SVM to work with ________ data.
4. SVM is a ________ learning algorithm.
5. C is a regularization parameter that controls the trade-off between ________ and margin.

## True or False

1. SVM can only use linear kernels.
2. Support vectors are the closest points to the decision boundary.
3. SVM is sensitive to feature scaling.
4. SVM can handle multi-class classification.
5. Larger C value leads to larger margin.

## Multiple Choice

1. What does a larger C parameter do?
   a) Larger margin, more misclassifications allowed
   b) Smaller margin, fewer misclassifications allowed
   c) No effect on the model
   d) Always gives better results

2. Which kernel is used for non-linear data?
   a) Linear
   b) RBF (Radial Basis Function)
   c) Polynomial
   d) Both b and c

3. What is the hyperplane in SVM?
   a) The decision boundary
   b) The margin
   c) The support vectors
   d) The kernel

## Answers

**Fill in the Blanks:**
1. hyperplane
2. support
3. non-linear
4. supervised
5. misclassification

**True or False:**
1. False - Can use RBF, polynomial, etc.
2. True
3. True
4. True
5. False - Smaller C = larger margin

**Multiple Choice:**
1. b
2. d
3. a
