# Deep Learning Algorithms Repository

This repository contains comprehensive implementations and explanations of key deep learning algorithms, organized folder-wise for easy learning and reference.

## Structure

Each algorithm folder contains:
- **README.md**: Detailed explanation with examples, use cases, limitations, benefits, and visualizations
- **Interview.md**: Most frequently asked interview questions with answers
- **Quiz.md**: Quiz based on important concepts
- **codes/**: Implementation examples including in-memory examples, CSV examples, exercises, and solutions
- **datasets/**: Required dataset files

## Algorithms Covered

1. **Artificial Neural Networks (ANN)**
2. **Convolutional Neural Networks (CNN)**
3. **Recurrent Neural Networks (RNN)**
4. **Long Short-Term Memory (LSTM)**
5. **Gated Recurrent Units (GRU)**
6. **Autoencoders**
7. **Variational Autoencoders (VAE)**
8. **Generative Adversarial Networks (GAN)**
9. **Transformer Networks**
10. **BERT (Bidirectional Encoder Representations from Transformers)**
11. **ResNet (Residual Networks)**
12. **U-Net**
13. **Siamese Networks**
14. **Capsule Networks**
15. **Neural Style Transfer**

## Getting Started

Each algorithm is designed to be self-contained and includes:
- Beginner-friendly explanations
- Mathematical foundations
- Practical implementations
- Real-world applications
- Interview preparation materials

## Prerequisites

- Python 3.7+
- TensorFlow/Keras or PyTorch
- NumPy, Pandas, Matplotlib
- Jupyter Notebook (recommended)

## Usage

Navigate to any algorithm folder to find complete learning materials and implementations.
