# Text Summarization - Quiz

## Q1: Extractive summarization does?
A) Generates new sentences
B) Selects sentences from original
C) Translates text
D) None

**Answer: B**

## Q2: Abstractive summarization does?
A) Copies from original
B) Generates new sentences
C) Classifies text
D) Translates

**Answer: B**

## Q3: ROUGE is used for?
A) Translation
B) Summarization evaluation
C) Classification
D) NER

**Answer: B**

## Q4: Which is NOT a summarization model?
A) BART
B) T5
C) SVM
D) PEGASUS

**Answer: C**
