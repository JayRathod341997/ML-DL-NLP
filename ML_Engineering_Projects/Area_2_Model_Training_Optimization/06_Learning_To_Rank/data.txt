# Learning to Rank — Public Datasets

## Primary Dataset
# MS MARCO Passage Ranking (large-scale web search dataset)
# ~8.8M passages, ~1M queries, sparse binary relevance labels
# HuggingFace dataset: ms_marco, config="v1.1"
https://huggingface.co/datasets/ms_marco

## Classic L2R Benchmark (LETOR 4.0)
# MQ2007: 1,692 queries, 69,623 docs, 5 relevance levels
# MQ2008: 784 queries, 15,211 docs
# Features: 46 hand-crafted IR features per (query, doc) pair
# Download from Microsoft Research:
https://www.microsoft.com/en-us/research/project/letor-learning-rank-information-retrieval/

## TREC Deep Learning Track 2019/2020
# Official IR evaluation with NIST relevance judgments
# Built on top of MS MARCO
https://microsoft.github.io/msmarco/TREC-Deep-Learning-2019
