# Word Embeddings - Quiz

## Questions

### Q1: Dimensions of word embeddings?
A) Same as vocabulary size
B) Fixed (typically 100-300)
C) Unlimited
D) One

**Answer: B**

### Q2: Word2Vec was developed by?
A) Facebook
B) Stanford
C) Google
D) Microsoft

**Answer: C**

### Q3: Which handles OOV words best?
A) Word2Vec
B) GloVe
C) FastText
D) All equally

**Answer: C (subword embeddings)**

### Q4: King - Man + Woman = ?
A) Prince
B) Queen
C) Princess
D) Royal

**Answer: B**

### Q5: Embeddings vs one-hot?
A) Same
B) Dense vs sparse
C) Sparse vs dense
D) No difference

**Answer: B**

### Q6: Which is contextual embedding?
A) Word2Vec
B) GloVe
C) ELMO
D) FastText

**Answer: C**

### Q7: Analogy solving works because?
A) Random vectors
B) Vector arithmetic
C) Dictionary lookup
D) Pattern matching

**Answer: B**

### Q8: GloVe uses?
A) Neural networks
B) Co-occurrence statistics
C) Random projection
D) Hashing

**Answer: B**
