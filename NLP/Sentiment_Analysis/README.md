# Sentiment Analysis

## Explain Like I'm 5

When someone says something, we can tell if they are happy, sad, or angry!

Sentiment analysis is like teaching a computer to understand feelings in text:
- "I love this!" = Positive/Happy 😊
- "This is terrible!" = Negative/Sad 😢
- "It's okay" = Neutral 😐

## What is Sentiment Analysis?

Determining the emotional tone or opinion expressed in text.

### Types:
1. **Binary**: Positive/Negative
2. **Multi-class**: Positive/Negative/Neutral
3. **Emotion Detection**: Happy, Sad, Angry, Fear, etc.

## Where Used?

1. **Brand Monitoring**: What do customers think?
2. **Social Media**: Track public opinion
3. **Customer Reviews**: Product feedback
4. **Stock Market**: Predict sentiment impact
5. **Customer Support**: Priority routing

## Approaches:

1. **Rule-based**: Lexicons (positive/negative word lists)
2. **Machine Learning**: Naive Bayes, SVM
3. **Deep Learning**: LSTM, BERT

## Enterprise Example:

**Amazon** analyzes product reviews to identify:
- Satisfied customers
- Issues with products
- Product improvements needed

This drives 35% of Amazon purchases through recommendations!
