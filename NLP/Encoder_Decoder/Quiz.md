# Encoder-Decoder - Quiz

## Q1: Seq2Seq is used for?
A) Classification
B) Translation
C) Clustering
D) Regression

**Answer: B**

## Q2: Encoder creates?
A) Output sequence
B) Context vector
C) Classification
D) None

**Answer: B**

## Q3: Attention helps?
A) Speed
B) Focus on relevant input
C) Reduce size
D) Nothing

**Answer: B**

## Q4: Translation uses?
A) Encoder only
B) Decoder only
C) Both encoder and decoder
D) Neither

**Answer: C**

## Q5: Decoder generates?
A) Input
B) Output sequence
C) Features
D) None

**Answer: B**
