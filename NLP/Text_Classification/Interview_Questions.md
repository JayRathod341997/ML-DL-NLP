# Text Classification - Interview Questions

## Q1: What is Text Classification?

Automatically assigning categories to text documents.

## Q2: Types of text classification?

1. Binary: Two classes (spam/not spam)
2. Multi-class: Multiple classes (news categories)
3. Multi-label: Multiple labels per document

## Q3: Traditional approaches?

1. TF-IDF + ML classifiers (Naive Bayes, SVM, Logistic Regression)
2. Bag of Words + classifiers

## Q4: Deep learning approaches?

1. CNN for text
2. LSTM/GRU
3. Transformers (BERT fine-tuning)

## Q5: How to handle imbalanced classes?

1. SMOTE
2. Class weights
3. Oversampling/Undersampling
4. Use appropriate metrics (F1 instead of accuracy)
