# Model Monitoring & Drift Detection — Datasets

## Primary Dataset for Drift Simulation
# Amazon Reviews — use different product categories as distribution shift
# Train reference: "raw_review_All_Beauty" (2018-era reviews)
# Production simulation: same category but recent reviews (drift over time)
# HuggingFace: McAuley-Lab/Amazon-Reviews-2023
https://huggingface.co/datasets/McAuley-Lab/Amazon-Reviews-2023

## Concept Drift Simulation (domain shift)
# IMDB — movie sentiment (training distribution)
# HuggingFace: imdb
https://huggingface.co/datasets/imdb

# Yelp Reviews — restaurant sentiment (production distribution = domain shift)
# HuggingFace: yelp_review_full
https://huggingface.co/datasets/yelp_review_full

# Drift scenario:
# 1. Train a model on IMDB (or use pre-trained)
# 2. Profile reference distribution on IMDB test set
# 3. Run inference on Yelp test set (concept drift: different domain)
# 4. Compare prediction distributions → drift detected
