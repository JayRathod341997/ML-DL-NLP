# Transformers Quiz

## True/False

### Q1: Transformers use recurrence.
**Answer:** False - they use self-attention.

### Q2: Self-attention captures long-range dependencies.
**Answer:** True.

### Q3: Positional encoding is optional.
**Answer:** False - needed for position info.

### Q4: Multi-head attention is parallel.
**Answer:** True.

## Multiple Choice

### Q5: Attention complexity is:
A) O(n)  B) O(n²)  C) O(log n)  D) O(1)
**Answer:** B

### Q6: Transformer was introduced in:
A) 2017  B) 2015  C) 2019  D) 2020
**Answer:** A
