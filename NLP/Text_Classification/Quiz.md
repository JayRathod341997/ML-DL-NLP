# Text Classification - Quiz

## Q1: Text classification assigns?
A) Numbers to text
B) Categories to documents
C) Images to text
D) None

**Answer: B**

## Q2: Binary classification has?
A) 1 class
B) 2 classes
C) 3 classes
D) Many classes

**Answer: B**

## Q3: Which is traditional approach?
A) BERT
B) TF-IDF + SVM
C) LSTM
D) Transformer

**Answer: B**

## Q4: Multi-label means?
A) One label per document
B) Multiple labels per document
C) No labels
D) Random labels

**Answer: B**

## Q5: Gmail uses classification for?
A) Translation
B) Spam detection
C) Summarization
D) None

**Answer: B**
